## Descripción general del Proyecto
EspressoInYourHeart es una plataforma web diseñada para conectar personas a través de su pasión por el café, generando coincidencias basadas en preferencias cafeteras, estilos de consumo, lugares favoritos y experiencias compartidas.

### El proyecto integra:

- Aplicación Web (Landing + Aplicación interna)

- Backend en Spring Boot

- Base de datos PostgreSQL

- UI/UX profesional con Figma

- Metodología Lean UX + Ágil (Scrum)

Este proyecto combina gastronomía, tecnología y diseño de experiencia para crear un ecosistema social y comercial centrado en la cultura del café.

## Objetivo del producto

### Para consumidores

- Facilite relaciones auténticas basadas en gustos cafeteros.

- Permita descubrir nuevas cafeterías y eventos.

- Genere conversaciones naturales usando el café como punto en común.

### Para propietarios de cafeterías

- Atraiga nuevos clientes.

- Permita organizar, promocionar y gestionar eventos.

- Aumente la visibilidad del negocio y su comunidad.

## Características principales
### Gestión de usuarios

- Registro/Login de consumidores y propietarios.

- Perfiles con preferencias cafeteras, edades, gustos y hábitos.

- Modificación de perfil, foto, contraseña, descripción.

### Gestión de cafeterías

- Registrar cafeterías con:
nombre, descripción, dirección, teléfono, horario, aforo.

- Listado de cafeterías y filtros por preferencias.

### Eventos

- Registro de nuevos eventos con estado, cantidad de participantes y fechas.

- Filtrado por cafetería.

- Visualización por rango de fechas.

- Listado por nombre de cafetería.

### Relación con consumidores

- Marcar cafeterías como favoritas.

- Reseñas, comentarios y preferencias detalladas.

- Listar consumidores que gustan de cada cafetería.

### Búsquedas y filtros

- Buscar cafeterías por preferencias del consumidor.

- Buscar eventos por cafetería.
